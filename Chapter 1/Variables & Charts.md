# Variables
## Categorical Variables
非数字 || 平均数无意义（比如手机号）

## Quantitative Variable
平均数有意义

# Charts
## Pie Chart
Better for Categorical Variables.

## Bar Chart
Better for Quantitative Variables.

## "Why is it not appropriate to use a pie chart for quantitative variables?"
Making a pie chart to display these data is not appropriate because each percent in the table refer to a different group, but not to parts of a single whole.
