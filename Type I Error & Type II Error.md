If we reject $H_{0}$ when $H_{0}$ is true, we have committed a **Type I error**.
If we failed to reject $H_{0}$ when $H_{a}$ is true, we have committed a **Type II error**.

|                          |                     |                      |
| ------------------------ | ------------------- | -------------------- |
|                          | $H_{0}$ is TRUE     | $H_{a}$ is TRUE      |
| Reject $H_{0}$           | <u>Type I Error</u> | Power                |
| Failed to reject $H_{0}$ | OK                  | <u>Type II Error</u> |
